# Mobile Initiative Archive

This folder keeps dated snapshots of superseded mobile initiative docs.

## Rules

- Each snapshot lives under a dated folder such as `2026-03-19/`.
- Keep the original area shape inside each snapshot, for example `root/`, `architecture/`, or `product/`.
- Archived docs are preserved for project history and should not be treated as the current operating source of truth.
- Do not rewrite archived content except to add minimal governance metadata for indexing and append archive metadata such as source path, archive date, and original file timestamps.
